# Task Manager – Full Stack Web Application

A **Full Stack Task Manager Web Application** built using **Spring Boot (Backend)** and **React.js (Frontend)**.
The application allows users to create, view, update, and delete tasks while storing the data in a **MySQL database**.

---

## 🚀 Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL

### Frontend

* React.js
* Axios
* Bootstrap
* HTML5
* CSS3
* JavaScript

### Tools & Technologies

* Git
* GitHub
* Postman
* VS Code
* Maven

---

## ✨ Features

* Create new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* REST API integration between frontend and backend
* Data persistence using MySQL database
* Responsive user interface

## ⚙️ How to Run the Project

### 1️⃣ Run Backend (Spring Boot)

Navigate to backend folder:

```
cd taskmanager
```

Run the Spring Boot application:

```
mvn spring-boot:run
```

Backend will start at:

```
http://localhost:8080
```

---

### 2️⃣ Run Frontend (React)

Navigate to frontend folder:

```
cd taskmanager-frontend
```

Install dependencies:

```
npm install
```

Start the React application:

```
npm start
```

Frontend will start at:

```
http://localhost:3000
```
